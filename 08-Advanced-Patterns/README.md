## When to Use Advanced Patterns

These patterns are for when:
- You're handling the same decision logic repeatedly (→ build an agent)
- Your team wants access to your tools (→ use the rollout framework)
- You want actions to happen automatically without manual triggers (→ hooks)
- You need your AI assistant to access external systems (→ MCP servers)

None of these are required for the core methodology. They're force multipliers for people who want to go further.
