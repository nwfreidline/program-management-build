# PMI Templates & Tools

> Industry-standard project management templates aligned to the PMBOK Guide and PMI methodology.
> Organized by project lifecycle phase: Initiating → Planning → Executing → Monitoring → Closing.

---

## Overview

These templates provide the formal documentation structure expected in professional project management. Use them as-is for formal projects, or adapt them for lightweight program tracking.

Each template is available in Markdown (for Obsidian use) and where applicable, Excel or Word format for stakeholder distribution.

---

## How to Use These Templates

### In Obsidian
1. Copy the desired template into your vault
2. Fill in the bracketed `[placeholder]` fields
3. Link to related project notes using `[[wikilinks]]`

### Standalone
1. Open the `.md` file in any text editor
2. Fill in the fields
3. Convert to Word/PDF using DocForge (included in this package) if needed for stakeholder distribution

### With AI Assistant
Ask your AI assistant:
> "Create a project charter for [project name] using the template in 07-PMI-Templates/Initiating/"

The AI will read the template structure and generate a populated draft for your review.
