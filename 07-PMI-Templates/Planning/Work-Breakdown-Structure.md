## WBS Guidelines

### Decomposition Rules
- Each level provides more detail than the level above
- Work packages are the lowest level and represent assignable, estimable work
- Apply the **100% Rule**: child elements must represent 100% of the parent's scope
- Work packages should typically be **8–80 hours** of effort
- Each element should have a single owner

### Numbering Convention
- Level 1: X.0 (Project)
- Level 2: X.X (Phase/Major Deliverable)
- Level 3: X.X.X (Work Package)
- Level 4: X.X.X.X (Activity — if further decomposition needed)
